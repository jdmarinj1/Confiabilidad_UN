Confiabilidad de sistemas eléctricos

Introducción

La confiabilidad, en su sentido más amplio, se relaciona con la capacidad de un sistema para resistir fallos, ofrecer un desempeño constante y generar confianza en su operación. No obstante, en el contexto de los sistemas eléctricos, la confiabilidad trasciende lo abstracto para convertirse en una disciplina técnica rigurosa, en la que es posible medir, planear y diseñar el comportamiento esperado de los equipos y redes ante diversas condiciones de operación.

Este libro (curso) aborda la confiabilidad como una herramienta fundamental en el diseño, operación y mantenimiento de los sistemas eléctricos, con especial énfasis en los sistemas de distribución y las instalaciones solares fotovoltaicas. Estos sistemas, por su cercanía al usuario final y su creciente integración con fuentes de energía distribuida, representan un nuevo desafío para los esquemas tradicionales de confiabilidad.

Históricamente, el sistema eléctrico se ha segmentado en tres dominios funcionales: generación, transmisión y distribución. En este texto nos enfocamos en la última etapa —la distribución— por ser la más vulnerable a interrupciones y la más exigente en cuanto a calidad y continuidad del servicio. Asimismo, exploraremos la confiabilidad en sistemas fotovoltaicos, dada su expansión en entornos urbanos, rurales y de difícil acceso, donde su desempeño confiable es esencial para garantizar el suministro eléctrico.

Learning Objectives for Tutorial

At the end of the tutorial, participants will be able to:

  • Identify the main features that make Julia an attractive language for Data Science
  • Set up a Julia environment to run their data analysis
  • Efficiently handle datasets (even across different languages) through Tables.jl and Arrow.jl
  • Fit (generalized) linear mixed models with MixedModels.jl
  • Communicate across languages (Julia, R, python)

Intended audience and level: The tutorial is intended for any data scientist with experience in R and/or python who is interested in learning the attractive features of Julia for Data Science. No knowledge of Julia is required.

In preparation for the workshop

Participants are required to follow the next steps before the day of the workshop:

  1. Git clone the workshop repository: git clone https://github.com/crsl4/julia-workshop.git

  2. Install Julia. The recommended option is to use JuliaUp:

  • Windows: winget install julia -s msstore

  • Mac and Linux: curl -fsSL https://install.julialang.org | sh

  • Homebrew users: brew install juliaup

After JuliaUp is installed, you can install different Julia versions with:

juliaup add release  ## installs release version
juliaup add rc       ## installs release candidate version
juliaup st           ## status of julia versions installed
juliaup default rc   ## make release candidate version the default
  1. Choose a dataset along with a script to analyze it written in another language (R or python) as we will spend part of the workshop translating participants’ scripts to Julia.
